# E1.4 Manual Integration Guide

## Step 1: Add Scenario Panel HTML to test.html

Open `viewer/test.html` and find line ~180 where the control-panel div ends:
```html
    </div>

    <script type="importmap">
```

**INSERT** the contents of `viewer/scenarioPanel.html` between these lines (after the control-panel closing `</div>` and before the `<script type="importmap">`).

## Step 2: Add ScenarioConfigUI Import to test.html

In the main `<script type="module">` block (around line 191), add this import at the top:

```javascript
import { ScenarioConfigUI, ScenarioSchemas } from './scenarioConfig.js';
```

## Step 3: Initialize ScenarioConfigUI

At the end of the main script block (after all existing viewer initialization, around line 550+), add:

```javascript
// E1.4: Initialize Scenario Configuration Panel
if (document.getElementById('scenarioPanel')) {
    try {
        new ScenarioConfigUI(
            document.getElementById('scenarioPanel'),
            ScenarioSchemas
        );
        console.log('[E1.4] Scenario Config Panel initialized');
    } catch (error) {
        console.error('[E1.4] Failed to initialize Scenario Config Panel:', error);
    }
}
```
